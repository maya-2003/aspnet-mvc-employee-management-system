# Employee & Department Management System (ASP.NET MVC)

## Overview
A web-based Employee and Department Management System built using ASP.NET MVC.  
The application follows a layered architecture (Presentation, Business Logic, Data Access) to ensure separation of concerns, scalability, and maintainability.

This project demonstrates enterprise-style design practices including repository pattern, service layer abstraction, and DTO-based data handling.

---

## Features
- Manage Employees (Create, Read, Update, Delete)
- Manage Departments (Create, Read, Update, Delete)
- User Authentication and Account Management
- Layered architecture (DAL, BLL, Presentation)
- Repository and Service pattern implementation
- Data transfer using DTOs
- Clean and structured codebase

---

## Architecture
The application is structured into multiple layers:

- **Presentation Layer (MVC)**  
  Handles UI, controllers, and user interaction

- **Business Logic Layer (BLL)**  
  Contains services, business rules, and application logic

- **Data Access Layer (DAL)**  
  Handles database operations using repositories and models

This separation improves maintainability and supports scalability.
